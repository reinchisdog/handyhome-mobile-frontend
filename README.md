# HandyHome: A Machine Learning-Enhanced Platform for Home Service Matching and Workforce Coordination

## Overview

HandyHome is a mobile app built to connect users with trusted home service professionals. The platform allow users to book professionals automatically, and/or solve problems on their own using a **built-in chat bot**. Meanwhile, this also allow workers to find jobs that they are able to render.

On another note, this is a front-end repository built with **React Native and Expo** provided to support the backend: https://github.com/darkttyu/handyhome-mobile-backend
